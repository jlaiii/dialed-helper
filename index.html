Installation

1. Install Tampermonkey or Violentmonkey for your browser

2. Click Install Userscript above — Tampermonkey detects it automatically

3. Go to dialed.gg and play. The helper panel appears in the bottom-right.

Features

Auto-Capture

Detects and records target colors from the countdown screen in real-time.

Set Color

Moves sliders to target without submitting — you press done when ready.

Auto-Solve

Sets sliders and auto-submits each round for full hands-free play.

Accuracy Modes

Perfect, Pro, or Good — controlled noise keeps scores believable.

Color Memory Panel

Shows captured colors with hex codes and HSB values for reference.

React Fiber Integration

Reads exact target values from the game's internal state for 100% precision.
